Refined Wordmarks

Score Wordmark

Orchestration Node Wordmark

Wordmark + Node Prefix

A compact orchestration node precedes the wordmark. Construction: outer circle (r=8px, 2.2px stroke), inner filled circle (r=2.2px, oxide). Score + Node Combined

Merges both approaches: the score lines beneath the wordmark, but the lower oxide rule terminates in a proper orchestration node (outer ring + inner fill) instead of a simple dot. Symbol Marks

Standalone Icons

Each symbol built on a 96×96 grid with 8px safe area. Consistent 2.2px strokes, grid-aligned angles, proper node construction. Shown at hero size (96px), nav size (32px), and favicon size (16px).
S1A

S-Curve — Smooth Flow

The letter S encoded as an orchestration signal path. Two arc-connected curves with a primary node (oxide, upper) and secondary node (mineral, lower). Entry tick marks the start of the flow.

S1B

S-Curve — Angular Grid

Same S-path concept, but built from strict 90° and 45° segments. More technical, systematic feel — like a PCB trace or railway switch diagram. Sharper personality while sharing the same DNA.

S3A

Ledger Block — Open

Vertical register with three internal oxide/mineral ticks and horizontal output stems of varied lengths. The tick hierarchy (two oxide + one mineral) mirrors the brand's three-color signal system. Output lengths encode data hierarchy.

S3B

Ledger Block — Compact

Tighter variant: five internal ticks at alternating weights (major oxide, minor ghost, major oxide, minor ghost, mineral) create a rhythm inside the block. Single oxide output stem at midpoint. More self-contained — stronger as a favicon.

S1C

S-Curve — Minimal

Maximum reduction: one continuous S-shaped curve with a single centered node at the inflection point. The node is the moment of transformation — complexity pivots to clarity. Elegant at every size. Strongest favicon candidate.

S3C

Ledger Rail — Vertical

The Data Rail motif expressed as a standalone icon. Three stops along a vertical rail (oxide primary, oxide secondary, mineral tertiary) with horizontal evidence stems. Directly references the brand's motif system. The rail is the spine of orchestration.
