# STROMY — Brand Guidelines

**Intelligence, Orchestrated.**

## 1. Brand Foundation

### Brand Essence
STROMY orchestrates AI-powered intelligence workflows so knowledge-intensive businesses can operate with the precision of a world-class ensemble.

### Positioning
For corporates, agencies, and consulting firms operating in knowledge-heavy domains, STROMY is the B2B partner that designs complex analytical workflows and builds AI tooling — delivering the structured intelligence infrastructure that transforms how organizations research, analyze, and decide.

### Tagline
**Intelligence, Orchestrated.**

### Brand Personality
- **Precise** — Every output is considered, structured, defensible.
- **Authoritative** — We speak from deep knowledge, not surface opinions.
- **Resourceful** — Complex problems are what we exist for.
- **Unflinching** — We deliver conclusions, not hedged possibilities.
- **Grounded** — No hype, no jargon theater, just results.

### Brand Voice
Write like someone who has read the briefing, done the analysis, and is giving you the conclusion — not selling you on the journey. Short sentences. Active voice. No filler words. Technical confidence without jargon theater.

## 2. Logo

### The Mark
The STROMY logo consists of the wordmark set in Instrument Serif, with a score line running beneath — a horizontal rule terminated by a signal orange dot. The score line references a musical staff (the maestro origin of the name) and doubles as a structural element across all brand touchpoints.

### Logo Anatomy
1. **Wordmark**: "STROMY" in Instrument Serif, letter-spacing 0.04em
2. **Score line**: 1.5px horizontal line in Stromy Green, running the full width of the wordmark
3. **Terminal dot**: 5px circle in Signal Orange, positioned at the end of the score line

### Lockup Variants
- **Primary**: Wordmark + score line (horizontal)
- **With tagline**: Primary + "INTELLIGENCE, ORCHESTRATED." beneath in DM Sans overline
- **Icon mark**: "S" in Instrument Serif on green rounded square + score line + dot
- **Monochrome**: Single-color version for emboss, watermark, or print constraints
- **Minimum size**: Score line drops below 70px width; only "S" + dot at 20px

### Logo Colors by Background
| Background      | Wordmark     | Score line      | Dot            |
|----------------|-------------|-----------------|----------------|
| Obsidian       | Parchment   | Stromy Green    | Signal Orange  |
| Parchment      | Obsidian    | Stromy Green    | Signal Orange  |
| Stromy Green   | Parchment   | Verdant (#3A7D65) | Signal Orange |
| White          | Obsidian    | Stromy Green    | Signal Orange  |

### Clear Space
Minimum clear space around the logo equals the cap-height of the "S" in the wordmark on all sides.

### Don'ts
- Never rotate or skew the logo
- Never change the colors of individual elements
- Never separate the dot from the score line
- Never add effects (shadows, glows, outlines)
- Never place on busy photographic backgrounds without a solid backing

- **Parchment** replaces pure white. Reports and light backgrounds use #F2F0EA, not #FFFFFF.
- **Obsidian** replaces pure black. Dark backgrounds use #0B0B0B.

## 5. The Score Line System

The score line is STROMY's signature motif — derived from the logo, it extends into a system-wide structural element.

### Variants
| Context           | Line width | Dot size | Line color                  |
|-------------------|-----------|----------|------------------------------|
| Logo              | Full wordmark width | 5px | Stromy Green              |
| Report cover      | Full page width     | 5px | Stromy Green              |
| Section divider   | 60px fixed          | 4px | Stromy Green              |
| Page footer       | Full content width  | 4px | Stromy Green (or Verdant) |
| Email signature   | 120px fixed         | 3px | Stromy Green              |
| Data separator    | Column width        | 3px | Stromy Green              |

### CSS Implementation
```css
.score-line {
  display: flex;
  align-items: center;
}
.score-line::before {
  content: '';
  flex: 1;
  height: 1.5px;
  background: var(--stromy-green-700);
}
.score-line::after {
  content: '';
  width: 5px;
  height: 5px;
  border-radius: 50%;
  background: var(--stromy-signal-orange);
}
```

## 6. Imagery Direction

### Photography Style
- **Urban, raw, material** — concrete, steel, glass, industrial textures
- **No nature photography** — no trees, landscapes, sunsets
- **No stock corporate** — no handshakes, no people smiling at laptops
- **Desaturated, high-contrast** — pull toward the neutral palette
- **Subjects**: building facades, construction materials, infrastructure, data centers, workshop tools, architectural details, raw textures (stone, metal, wood grain)

### Treatment
- Apply a slight green (#1B3D33) color overlay at 15-20% opacity for brand cohesion
- Crop tight — architectural details, not full buildings
- Prefer symmetry and geometric composition

### Usage
- Website hero backgrounds (with dark overlay for text legibility)
- Report cover full-bleed images
- Slide divider pages
- Section backgrounds with overlay

## 7. Iconography

### Style
- **Stroke-based**, 1.5px weight
- **Rounded caps and joins**
- **24×24px grid**, optically centered
- Color: Stromy Green on light backgrounds, Parchment on dark backgrounds
- Source recommendation: Lucide Icons (matches weight and style)

### Don'ts
- No filled icons (except for active/selected states)
- No multicolor icons
- No illustration-style icons

## 8. Brand Applications

### Report Covers
- Obsidian background
- "STROMY INTELLIGENCE" overline in Verdant
- Title in Instrument Serif, Parchment
- Full-width score line beneath title
- Date and classification in Pewter

### Presentation Slides
- Title slide: Stromy Green background, Instrument Serif title
- Content slides: Parchment background, DM Sans body
- Score line as footer rule on every slide
- Small logo lockup bottom-left

### Email Signatures
- Score line (120px, short variant) above name
- Name: DM Sans Medium, 13px
- Title + company: DM Sans, 11px, Stone
- URLs: IBM Plex Mono, 10px

### Website
- Hero: Obsidian background, centered logo with tagline
- Section transitions: score line dividers
- Cards: Parchment background, subtle border
- CTAs: Stromy Green primary, Signal Orange for emphasis
